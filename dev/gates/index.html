Gates

BraketSimulators.jl provides many pre-implemented gates which can be used to build up circuits. For gates with angle parameters, you can supply Irrationals like π as arguments.

BraketSimulator.Gate — Type
Gate <: QuantumOperator

Abstract type representing a quantum gate.

BraketSimulator.AngledGate — Type
AngledGate{NA} <: Gate

Parametric type representing a quantum gate with NA angle parameters.

BraketSimulator.GPhase — Type
GPhase{N}(angle)

Global phase shift on N qubits. Equivalent to the OpenQASM3 built-in gphase gate. Controls/negative controls applied to this gate control which states are rotated, so that Control(g::GPhase{2}) will apply the rotation to the |11> state.
